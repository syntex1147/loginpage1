Outputs and Use

Patient Progression: The chatbot will act as a therapist, and update you with any significant progressions. Whether this means more physical meetings, or medicine administration, the use of the chatbot is to give you insights of the patient.

Media Upload and Diagnosis: Our model takes in a .wav file (audio), a .mat file (eeg), and text input to diagnose the patient, and create the case-specific chatbot. Our model is very accurate, but is only used for initial setup and diagnostics (it can be used for intermediate checkups if needed).

Multi-Headed CNN Model

Multi-headed CNN, or Multi-headed Convolutional Neural Networks, have been extremely useful for multimodal data and image analysis. We have adapted the various types of data in our process (eeg, questions, audio) into tabular and image data through the Mel spectrogram and Fourier transform.

The three branches of this complex ML architecture allow the model to analyze all three forms of patient input, and accurately predict mental-health conditions for patients. The use of a CNN approach allows for the intricate patterns and features of eeg/voice image data to be analyzed, yielding greater prediction/diagnosis accuracies. The model is trained on a large dataset for audio/eeg/questions, for 5 ethnicities from 2 countries. The chatbot is also trained on a large vector database full of medical/psychological data, so the chatbot LLM can accurately emulate a real therapist.
